# 🧭 Contos Perdidos — Mundo de Zarton

Este projeto é um site desenvolvido para apresentar o universo fictício de **Zarton**, criado para campanhas de RPG.

O objetivo do site é organizar informações sobre reinos, personagens e narrativa, servindo como apoio para jogadores e mestres.

---

## ⚠️ Aviso Importante

Este projeto **não possui fins lucrativos**.

* Nenhum conteúdo aqui é utilizado para gerar renda
* O site foi criado apenas para uso pessoal, educativo e narrativo
* Não há qualquer tipo de monetização envolvida

---

## 🎨 Uso de Imagens e Artes

As imagens utilizadas neste projeto:

* **Não foram criadas por mim**
* Foram obtidas da internet ou geradas por ferramentas de terceiros
* São utilizadas apenas para fins ilustrativos dentro do projeto

Caso alguma imagem utilizada seja de sua autoria e você deseje a remoção ou crédito adequado, entre em contato.

---

## 📜 Direitos e Intenção

Este projeto é uma obra **fanmade / autoral de entretenimento**, sem qualquer intenção de violar direitos autorais.

Todo o conteúdo foi reunido com o objetivo de:

* Criar uma experiência imersiva de RPG
* Compartilhar um universo fictício
* Servir como material de apoio para campanhas
